## 🫀 Heart Disease Prediction using Machine Learning

This project builds a machine learning model to predict whether a person has heart disease based on medical attributes.

---

## 📌 Project Overview

The objective of this project is to analyze patient health data and develop a predictive model that can classify whether a person is likely to have heart disease.

The project includes data preprocessing, data analysis, model training, and evaluation.

---

## 📊 Dataset Information

* Total Records: **303**
* Total Features: **13 + 1 Target**
* No missing values in dataset

### Features include:

* Age
* Sex
* Chest Pain Type (cp)
* Resting Blood Pressure (trestbps)
* Cholesterol (chol)
* Fasting Blood Sugar (fbs)
* Maximum Heart Rate (thalach)
* Exercise Induced Angina (exang)
* Oldpeak
* Slope, CA, Thal

### Target Variable:

* **0 → No Heart Disease**
* **1 → Heart Disease**

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn

---

## 🔍 Project Workflow

1. Data Collection & Loading
2. Data Preprocessing
3. Data Analysis
4. Splitting Features & Target
5. Train-Test Split (80% training, 20% testing)
6. Model Training using Logistic Regression
7. Model Evaluation

---

## 🤖 Model Used

* Logistic Regression

---

## 📈 Model Performance

* **Training Accuracy:** 85.12%
* **Testing Accuracy:** 81.96%

---

## 🧠 Predictive System

The model takes input data and predicts whether a person has heart disease or not.

Example:
Input: [62,0,0,140,268,0,0,160,0,3.6,0,2,2]
Output: No Heart Disease

---

## 📌 Key Insights

* Dataset is clean with no missing values
* Logistic Regression gives good performance
* Model generalizes well on test data
